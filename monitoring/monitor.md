# Monitoring

In monitoring, we are going to check which process are consuming more cpu, memory, disk. there are commands to help admin top process which consuming more amount of memory and cpu. 

command list:

cpu and memory monitoring:
1. top = it show real time data which keep updating every second. it is listed in order by order.
2. htop = show more beautifully with color in user interface
3. vmstat = it report system performanceand you check a lot of free memory or not. for instance, someone is complainig that your linux instance is slow so you can use vmstat. 
4. free -m = it shows free memory, top is also used but for script only free command is used.

disk monitoring:
1. nproc = amount of cpu avialable
2. df -h = to check disk space
3. du -sh = folder utilization
