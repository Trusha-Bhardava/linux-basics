#Networking

networking is very important for debuggind,troubleshooting. 

important command (use in script):

1. ip -a = show ip address of network interface.
2. ping = check connectivity to remote server. if you see 100% packet loss then webite unsuccesful connection
3. curl = fetch webite data
4. wget = download file
5. hostname = system name
6. ifconfig = display network interface
