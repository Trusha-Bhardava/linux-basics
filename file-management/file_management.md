# linux file management

This script demonstrate basic file operation like:

1. create file and folder
2. list file and folder
3. copy file 
4. rename file name
5. delete file and folder

Now there are more commands related to file viewing and editing,

1. cat filename - Display file content
2. less filename - opens file content with scrolling support
3. more filename - similar to less, buit only moves forward
4. head -n 10 filename - show start 10 line
5. tail -n 10 filename - show last 10 line
6. vi filename - open powerful text editor
7. echo "hello" > filename - write text to file, overwriting
8. echo "hi" >> filename - appends text without overwriting
9. wc filename - shows line,word,bytes in order for know how file big
10. diff file1 file2 - show difference between file
